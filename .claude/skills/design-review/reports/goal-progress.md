# /goal — прогресс полного дизайн-ревью курса

Автономный проход по всем 57 урокам через скилл design-review (все 7 линз, включая дизайн-полиши).
Единица = 1 урок за ход: capture → 7 линз → правки в границах (a11y/визуал сразу; текст/simple с суждением) → коммит на урок → отметка ниже.
Границы правок: безопасное + текст уроков с суждением. Каждый инструментальный кандидат сверять глазом.

Легенда: [ ] не сделан · [x] отревьюен и закоммичен · (ок) — правок не потребовалось
