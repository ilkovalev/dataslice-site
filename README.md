# DataSlice — лид-магнит

Интерактивный сайт-лид-магнит для Telegram-канала DataSlice. Две страницы:

- **`/stats`** — интерактивный курс статистики для новичков: 12 модулей, 56 уроков с интерактивными графиками (от описательной статистики до A/B, регрессии, классификации и Байеса).
- **`/metrics`** — иерархии метрик по 15 индустриям (деревья North Star → драйверы → операционные → контр-метрики) + глоссарий бизнес-метрик.
- **`/glossary`** — глоссарий статистических и бизнес-терминов с поиском.

## Стек
React + Vite + Tailwind, react-router. Сайт полностью статический (вся интерактивность — в браузере), деплоится как набор статических файлов.

## Запуск
```bash
npm install
npm run dev      # дев-сервер
npm run build    # прод-сборка в dist/
npm run preview  # предпросмотр сборки
```

## Структура
```
src/
  pages/        StatsPage, MetricsPage, GlossaryPage
  components/   виджеты-интерактивы + реестр widgets.js; BeatsLesson (модель урока)
  content/      lessons/*.json — уроки; industries/*.json — деревья метрик (контент = данные)
  lib/          distributions.js — реестр распределений
```
