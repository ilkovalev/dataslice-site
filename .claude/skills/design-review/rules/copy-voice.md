# Линза: Копирайт / тон

Вход: текст урока из `src/content/lessons/<slug>.json`. Подчинён `clarity.md`.
Эталон тона — `feedback-dataslice-course-tone` + референс SFDA2022 (манера, без грубого сленга).
Severity по умолчанию — «важное» (тон — конкурентное преимущество), отдельные — «полиш».

---

## human-translation — определение → человеческий перевод
Строгая формулировка сопровождается переводом на человеческий.
- **Вход (инструментально):** поле `definitions[].simple` непусто и реально проще, чем `text`.
- Сигнал: определение подано без «ну, то есть…»-перевода; `simple` дублирует `text` канцеляритом.

## active-voice — активный залог, без канцелярита
- Сигналы: «необходимо упорядочить» вместо «упорядочим»; отглагольные существительные, пассив, нанизывание родительных падежей.

## reader-in-boat — читатель в одной лодке, на «Вы»
- Сигналы: длинный отрезок без обращения к читателю; «ты» вместо «Вы»; сухое изложение от третьего лица там, где нужно «давайте посмотрим».

## voice-calibration — калибровка манеры
Берём манеру SFDA2022, но приглушаем: грубый сленг («салат», «человеков») и избыток «я» — флаг.
- Сигналы: грубый сленг; переизбыток авторского «я»; ИЛИ обратный перекос — стерильный учебник без живого голоса.

## anti-ai-writing — без ИИ-паттернов
Прогонять через skill `avoid-ai-writing-ru`.
- Сигналы: «в мире X», «важно отметить», симметричные тройки, пустые связки, водянистые заходы.

---

**Плохо → Хорошо** (урок `center-measures`, поле определения):
- Плохо: «Медиана — значение признака в середине упорядоченного вариационного ряда. Для расчёта необходимо упорядочить выборку.» _(определение+инструкция, пассив, нет перевода)_
- Хорошо (реальный `simple`): «значение из середины ряда — редкие крайние точки его не сдвигают.»
