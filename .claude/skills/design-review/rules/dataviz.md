# Линза: Качество визуализации (dataviz)

Графики — ядро продукта DataSlice. Вход: скриншоты состояний (все вьюпорты, до/после
действия) + метрики. Содержание — skill `dataviz` (form heuristic, палитра-валидатор,
спеки осей/легенд) + механики Seeing Theory как **инструменты понятности**, не эталон копирования.
Верховный судья — `clarity.md` (поймёт ли новичок). Severity по умолчанию — «важное».

---

## chart-clarity — график объясняет концепцию с нуля (главное правило линзы)
График должен помогать понять идею урока, а не требовать её уже знать.
- **Вход:** скриншот + `novice-test` из `clarity.md`.
- Сигналы: чтобы прочитать график, надо заранее знать концепцию; график показывает результат, но не идею; визуальный акцент не на том, что объясняет бит (`beats[].widget.highlight` не совпадает с фокусом текста).

## sim-convergence — показать процесс, а не итог
Для тем оценивания / ЗБЧ / ЦПТ / выборок эмпирика **накапливается на глазах** и сходится
к явно показанному истинному значению-якорю.
- **Вход:** скрины до/после массового действия (× N).
- Сигналы: показан только финальный результат без накопления; истинное значение-якорь не показано как ориентир сходимости; для темы про сходимость нет анимации процесса.
- Движение — только осмысленное; уважать `prefers-reduced-motion` (см. `BRAND.md#3`).

## chart-encoding — осмысленный, консистентный цветовой код
Цвет снижает когнитивную нагрузку и различает сущности; одна сущность — один цвет **во всех уроках**.
- **Вход:** скрин + контраст пар серий из `capture`.
- Сигналы: «истинное значение» в одном уроке одного цвета, в другом — другого (разнобой роли); цвет декоративен, не кодирует; парные функции одного объекта (плотность/накопленная) не связаны родственными оттенками.

## chart-honesty — график не вводит в заблуждение
- **Вход:** скрин + диапазоны осей из `capture`.
- Сигналы: усечённая ось там, где нужна нулевая база; искажённые пропорции; двойные оси без явной необходимости.

## chart-legibility — читаемость
- **Вход:** скрины во всех вьюпортах (390/768/1440). Только светлая тема (`BRAND.md#3`).
- Сигналы: оси не подписаны; легенда-загадка; метки наезжают; на 390px график нечитаем/обрезан; интерактивные элементы графика меньше тап-минимума (см. `accessibility.md`).
