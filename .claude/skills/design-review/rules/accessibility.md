# Линза: Доступность

Почти полностью инструментальна (объективные метрики из `capture`). Только светлая тема (`BRAND.md#3`).
Severity: контраст текста и тап-зоны — **blocker/важное**; остальное — «полиш».

---

## a11y-contrast — контраст текст/фон по WCAG AA
- **Вход:** посчитанный контраст пар текст/фон из `capture`.
- Сигналы: основной текст < 4.5:1; крупный текст (≥24px/≥19px bold) < 3:1; серии графика/подписи < 3:1.
- Опорные точки из кода: `cyanink #0a6c97` заявлен как AA на бежевом `#FFF8EF` — проверить фактически.

## tap-targets — размер интерактивных зон
- **Вход:** размеры интерактивных элементов из `capture` (в мобильном вьюпорте).
- Сигналы: тап-зона < 44×44px (кнопки, точки графика, переключатели); соседние зоны без зазора.

## focus-visible — видимый фокус с клавиатуры
- Сигналы: интерактивный элемент без видимого фокус-кольца; порядок табуляции не следует смыслу.
- В коде задано `:focus-visible { outline: 2px solid #0a6c97 }` — проверить, что не переопределено локально.

## alt-text — альтернативы для смысловой графики
- **Вход:** наличие `alt`/`aria-label` из `capture`.
- Сигналы: смысловая иллюстрация без alt; декоративная — с шумным alt вместо пустого; интерактивный SVG без текстовой альтернативы сути.

## reduced-motion — уважение к prefers-reduced-motion
- Сигналы: анимация без ветки `@media (prefers-reduced-motion)`; критичная для понимания информация доступна только через движение.
